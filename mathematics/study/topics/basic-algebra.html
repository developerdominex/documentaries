1. Introduction

Algebra is a branch of mathematics that uses symbols (usually letters) to represent numbers and express general relationships. It allows us to describe patterns, solve problems, and express mathematical rules in a universal form.

Historical Background

Algebra originated in ancient Babylon, where equations were solved using geometric methods. The word “algebra” comes from the Arabic al-jabr, from a 9th-century book by Al-Khwarizmi titled Al-Kitab al-Mukhtasar fi Hisab al-Jabr wal-Muqabala (“The Compendious Book on Calculation by Completion and Balancing”).

2. Key Concepts

  • Variables: Letters that represent unknown or changeable quantities (e.g., x, y, z).
  • Constants: Fixed values (e.g., 2, −5, π).
  • Expressions: Combinations of numbers, variables, and operations (e.g., 2x + 3).
  • Equations: Statements that two expressions are equal (e.g., 2x + 3 = 7).
  • Inequalities: Statements comparing two expressions (e.g., x + 5 > 8).

3. Basic Operations in Algebra

Algebra follows the same arithmetic operations but uses variables.

Examples:

  • Addition: (3x + 2) + (2x − 5) = 5x − 3
  • Subtraction: (6x − 4) − (2x + 1) = 4x − 5
  • Multiplication: 3x × 2x = 6x²
  • Division: 6x² ÷ 3x = 2x

4. Laws and Properties

  • Commutative: a + b = b + a; a × b = b × a
  • Associative: (a + b) + c = a + (b + c)
  • Distributive: a(b + c) = ab + ac
  • Identity: a + 0 = a; a × 1 = a

5. Solving Linear Equations

A linear equation has the form ax + b = c, where a, b, and c are constants. Solving means finding the value of x that makes the equation true.

Example:

2x + 3 = 7

Subtract 3 from both sides → 2x = 4

Divide by 2 → x = 2

6. Quadratic Equations

A quadratic equation is of the form ax² + bx + c = 0.

Solutions (roots) are given by the quadratic formula:

x = (-b ± √(b² − 4ac)) / (2a)

Example: x² − 5x + 6 = 0 → x = 2 or 3

7. Inequalities

Inequalities express relationships like greater than or less than. Example: 3x + 2 > 8 → 3x > 6 → x > 2.

Note that when you multiply or divide both sides by a negative number, the inequality sign reverses.

8. Graphs and Functions

A function expresses a relationship between input (x) and output (y). Example: y = 2x + 3 is a linear function, producing a straight line on the graph.

If x = 0 → y = 3; If x = 1 → y = 5; If x = 2 → y = 7.

9. Simplifying and Factorization

Simplify by combining like terms and using algebraic rules.

Example: 3x + 5x − 2 = 8x − 2

Factorization means expressing an expression as a product of simpler terms.

Example: x² − 9 = (x + 3)(x − 3)

10. Word Problems

Algebra translates real-world problems into equations.

Example: A number increased by 7 equals 15. Find the number.

Let the number = x.

x + 7 = 15 → x = 8

11. Modern Applications

  • Computer algorithms and programming logic
  • Economics and finance modeling
  • Physics equations and motion problems
  • Engineering design and optimization

12. Summary

Algebra transforms arithmetic into a symbolic language that expresses patterns and rules. It forms the foundation for all higher mathematics — from geometry to calculus and beyond.
