Functions & Graphs

Understanding relationships between variables through equations and visual graphs.

What is a Function?

A function is a special relationship where each input has exactly one output. It’s like a machine: you put in an input (x), and it gives one output (y).

Example: f(x) = 2x + 3
Input x = 2 → Output f(2) = 2×2 + 3 = 7

Domain and Range

  • Domain: All possible input values (x-values) that can go into the function.
  • Range: All possible output values (y-values) that come out of the function.
For f(x) = √x → Domain = x ≥ 0, Range = y ≥ 0

Graph of a Function

The graph of a function is a visual representation of all (x, y) pairs. It helps to understand how outputs change with inputs.

Example: Graph of a Quadratic Function (y = x²)

Types of Functions

  • Linear Function: f(x) = mx + c (straight line)
  • Quadratic Function: f(x) = ax² + bx + c (parabola)
  • Cubic Function: f(x) = ax³ + bx² + cx + d
  • Exponential Function: f(x) = aᵡ
  • Trigonometric Function: f(x) = sin x, cos x, tan x

Transformations of Graphs

  • Translation: Moves graph up, down, left, or right.
  • Reflection: Flips graph over an axis.
  • Stretch/Compression: Changes the shape by scaling vertically or horizontally.
Example: y = (x - 2)² moves 2 units to the right. y = -x² reflects the parabola in the x-axis.

Composition of Functions

If you have two functions f(x) and g(x), the composition f(g(x)) means “apply g first, then f.”

Example:
f(x) = 2x, g(x) = x + 3 → f(g(x)) = 2(x + 3) = 2x + 6

Inverse of a Function

The inverse function reverses the operation of the original function. If f(x) = 2x + 3, then its inverse is f⁻¹(x) = (x - 3)/2.
