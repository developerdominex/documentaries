Matrices & Determinants

Matrix operations, inverses, determinants, and applications.

Introduction to Matrices

A matrix is a rectangular array of numbers arranged in rows and columns. Matrices are widely used in mathematics, physics, computer science, and engineering.

Example:
123
456
789
A 3×3 matrix

Matrix Operations

  • Addition: Sum of matrices of same size, add corresponding elements.
  • Subtraction: Subtract corresponding elements of matrices of same size.
  • Scalar Multiplication: Multiply every element by a constant.
  • Matrix Multiplication: Product of two matrices. Number of columns of first = number of rows of second.
  • Transpose: Rows become columns.
A = [[1,2],[3,4]], B = [[5,6],[7,8]]
A + B = [[6,8],[10,12]]
A × B = [[19,22],[43,50]]
Aᵀ = [[1,3],[2,4]]

Determinants

The determinant is a scalar value that can be computed from a square matrix. It has important properties in solving linear equations, finding inverses, and transformations.

2×2 Matrix:
|A| = |a b| = ad - bc
|c d|
Example: A = [[2,3],[1,4]] → |A| = (2×4) - (3×1) = 8 - 3 = 5
3×3 Matrix:
|A| = a(ei − fh) − b(di − fg) + c(dh − eg)

Inverse of a Matrix

The inverse of a square matrix A is denoted A⁻¹ such that A × A⁻¹ = I (identity matrix). Only square matrices with non-zero determinant have inverses.

A = [[1,2],[3,4]] → det(A) = -2 → A⁻¹ = (1/det(A)) * [[4,-2],[-3,1]] = [[-2,1],[1.5,-0.5]]

Applications of Matrices

  • Solving systems of linear equations
  • Computer graphics transformations
  • Cryptography
  • Markov chains and probabilities
  • Engineering simulations

History of Matrices

Matrices were used in China as early as 2nd century BC in solving systems of linear equations. The modern notation and study of determinants and matrix operations were developed in the 19th century by mathematicians like Cayley and Sylvester.
