Category Theory

Functors, natural transformations, and abstract structures

Introduction

Category theory studies mathematical structures and relationships abstractly. It emphasizes **morphisms** (arrows) between objects rather than the internal structure of objects themselves. It is a unifying language across mathematics, computer science, and logic.

Categories

A category consists of objects and morphisms (arrows) that satisfy:

  • Composition: For morphisms f: A → B and g: B → C, there exists g∘f: A → C
  • Identity: For each object X, there is an identity morphism id_X: X → X
  • Associativity: h∘(g∘f) = (h∘g)∘f for all composable morphisms
Example:
Objects: Sets
Morphisms: Functions between sets

Functors

A functor maps objects and morphisms from one category to another while preserving structure (composition and identities).

Example: The functor F: Set → Set that assigns to each set X its power set P(X) and to each function f: X→Y the function F(f): P(X) → P(Y), F(f)(S) = f(S).

Natural Transformations

A natural transformation provides a "morphism of functors", giving a systematic way to transform one functor into another while respecting the category structure.

Example: For functors F, G: C → D, a natural transformation η: F ⇒ G assigns each object X in C a morphism η_X: F(X) → G(X) in D such that for any f: X→Y in C, G(f)∘η_X = η_Y∘F(f)

Applications

  • Abstract algebra: unify groups, rings, modules
  • Topology: study homotopy and homology functors
  • Computer science: functional programming, type theory
  • Logic: categorical semantics of mathematical theories

History

Category theory was introduced by Eilenberg and Mac Lane in 1945. It has grown into a central framework for modern mathematics and theoretical computer science, providing a universal language for structure and abstraction.
