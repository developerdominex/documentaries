Set Theory

Axioms, ordinals, cardinals, and continuum hypothesis

Introduction

Set theory is the foundation of modern mathematics. It studies collections of objects, called sets, and their relationships. Set theory provides the language for nearly all mathematical concepts.

Basic Concepts

  • Set: A collection of distinct objects, e.g., A = {1,2,3}
  • Element: An object inside a set, e.g., 2 ∈ A
  • Subset: B ⊆ A if every element of B is in A
  • Union: A ∪ B, elements in A or B
  • Intersection: A ∩ B, elements in both A and B
  • Complement: A', elements not in A

Axiomatic Set Theory

To avoid paradoxes, modern set theory uses axioms (Zermelo-Fraenkel set theory with Choice, ZFC):

  • Axiom of Extensionality
  • Axiom of Pairing
  • Axiom of Union
  • Axiom of Infinity
  • Axiom of Power Set
  • Axiom of Choice (AC)

Ordinals

Ordinals extend natural numbers to describe the order type of sets:

0, 1, 2, …, ω (first infinite ordinal), ω+1, ω+2, … Ordinals describe position in well-ordered sets.

Cardinals

Cardinals measure the size of sets, ignoring order:

Finite sets: |{1,2,3}| = 3
Infinite sets: |ℕ| = ℵ₀ (aleph-null), |ℝ| = 2^ℵ₀ (cardinality of continuum)

Continuum Hypothesis

The Continuum Hypothesis (CH) asks: Is there a set whose cardinality is strictly between |ℕ| and |ℝ|?

Proven to be independent of ZFC: cannot be proven or disproven using standard axioms.

Applications

  • Foundations of mathematics
  • Logic and model theory
  • Computer science: data structures, formal languages
  • Analysis and topology: definitions of functions, sequences, and spaces

History

Developed by Georg Cantor in the late 19th century, set theory revolutionized mathematics and introduced concepts of infinity rigorously.
