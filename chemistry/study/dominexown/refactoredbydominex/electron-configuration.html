⚛️ Electron Configuration of Atoms

Overview

Electron configuration describes the arrangement of electrons in an atom's orbitals. Understanding it is essential to predict chemical properties, bonding patterns, and reactivity.

Aufbau Principle

The Aufbau principle states that electrons occupy the lowest energy orbitals first before moving to higher energy orbitals.

  • Order of filling: 1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s → 5f → 6d → 7p
  • Helps predict the electronic arrangement of elements in the periodic table

Pauli Exclusion Principle

No two electrons in an atom can have the same set of four quantum numbers. This ensures that each electron has a unique position in the atom.

Hund’s Rule

Electrons fill degenerate orbitals singly first, with parallel spins, before pairing up. This minimizes electron repulsion and stabilizes the atom.

  • Example: Carbon → 1s² 2s² 2p² → two unpaired electrons in separate p orbitals

Electron Configuration Notation

Electron configurations can be written in two ways:

  • Full notation: Lists each orbital with electron occupancy (e.g., Oxygen: 1s² 2s² 2p⁴)
  • Short-hand / Noble gas notation: Uses previous noble gas to represent filled orbitals (e.g., Oxygen: [He] 2s² 2p⁴)

Exceptions in Electron Configuration

Certain elements (transition metals, lanthanides, and actinides) have electron configurations that differ slightly due to stability of half-filled or filled orbitals.

  • Example: Chromium → [Ar] 3d⁵ 4s¹ instead of [Ar] 3d⁴ 4s²
  • Example: Copper → [Ar] 3d¹⁰ 4s¹ instead of [Ar] 3d⁹ 4s²

Orbital Diagrams

Orbital diagrams visually represent electron arrangements, showing electrons as arrows in boxes representing orbitals. This makes it easier to apply Hund’s rule and Pauli exclusion principle.
